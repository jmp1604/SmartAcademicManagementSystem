Template & Format Guide

Required Columns

  • Student ID — Unique 7-digit school ID (e.g. 2300221). Also used as default password.
  • First Name — Student's first name
  • Last Name — Student's last name

Optional Columns

  • Middle Name — May be left blank
  • Course — e.g. Computer Science
  • Year Level — Number from 1–5
  • Section — e.g. CS3A
  • Email — Auto-generated if blank: lastname_firstname@plpasig.edu.ph

Notes

  • Default password = Student ID (e.g. 2300221)
  • Duplicate Student IDs will update the existing record
  • Do not delete the header row
  • Remove sample data before importing
